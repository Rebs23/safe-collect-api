Authentication

All requests must be authenticated via the Authorization header using your SDF Private Key.

curl -X POST https://api.safe-collect.io/v1/charge \
  -H "Authorization: Bearer YOUR_SDF_KEY"

Error Handling

Our system uses autonomous error resolution. However, if a critical failure occurs, you will receive a standard HTTP code with a SDF reason.

  • 401 Unauthorized: Invalid API Key.
  • 409 Conflict: Idempotency Key already processed.
  • 504 Timeout: Gateway failure (Sidecar will auto-retry).
